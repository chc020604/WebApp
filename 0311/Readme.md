* SEO(검색엔진 최적화): 
    * 메타 태그를 많이 달아둘 수록 노출 빈도가 올라갈 수 있다.

* HTML Standard
    1. W3C (World Wide Web Consortium)
    2. MDN Web Docs (Mozilla Developer Network)
    3. WHATWG (Web Hypertext Application Technology Working Group)

* MPA와 SPA
    * MPA: 클릭해서 넘어가는 일반 웹 사이트 각각의 페이지가 만들어져있는 경우 (페이지 전체가 바뀌는 경우)
    * SPA: 

* 하나의 프로젝트가 만들어지면 하나의 디렉토리가 만들어진다.

* node.js: 서버 측에서 자바스크립트를 실행할 수 있게 해주는 런타임 환경

* Platform, Environment, Framework, Runtime
    1. Platform: 소프트웨어가 실행될 수 있는 기반 환경
    2. Environment: 특정 소프트웨어가 실행될 수 있도록 구성된 조건 
    3. Framework: 소프트웨어 개발을 위한 뼈대 또는 기반 구조
    4. Runtime: 프로그램이 실행되는 동안 필요한 환경 
    **node.js에서의 위 4가지의 의미가 조금 다르다.**

* IDE: Integrated Development Environment
**vs code는 기본적으로 편집기이며 확장 프로그램을 통해 IDE역할을 할 수 있다.**

* npm, npx
    1. npm: Node.js 패키지를 설치, 관리하는 패키지 매니저
    2. npx: npx는 npm 패키지를 실행하는 도구

* 리액트 프로젝트를 만들기 위해 vite또는 Create React App(CRA)를 사용할 수 있다.
